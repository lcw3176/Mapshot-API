== 공지사항

=== 목록 조회

공지사항 목록을 조회합니다.
세부적인 공지사항의 내용은 전달되지 않습니다.
전달받은 페이지 번호로 최대 10개의 목록을 반환하며, 페이지 번호를 0번으로 요청 시 가장 최신 게시글 10개의 목록이 반환됩니다.
